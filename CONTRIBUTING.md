# Contributing Guidelines

Thank you for your interest in contributing to this repository.

This framework is curated to preserve governance integrity,
safety standards, and operational consistency.

---

##  Accepted Contributions

We welcome contributions that:

- Improve documentation clarity
- Enhance structural organisation
- Fix typographical or formatting issues
- Refine governance processes
- Update compliance references

---

##  Not Accepted

The following will not be accepted:

- Removal of safety or compliance controls
- Weakening of governance procedures
- Unreviewed policy changes
- Informal or unsupported practices

---

##  Contribution Process

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Submit a Pull Request
5. Address review feedback
6. Merge after approval

All major changes require maintainer review.
